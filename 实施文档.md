## 📈 第四阶段：统计分析（Pandas 核心）

### 4.1 计算总分和平均分

**任务**：新增两列，分别计算每个人的总分和平均分。

**代码指引**：

```
# 计算总分（axis=1 表示按行相加）
df['total_score'] = df[['chinese', 'math', 'english', 'physics']].sum(axis=1)

# 计算平均分
df['average'] = df[['chinese', 'math', 'english', 'physics']].mean(axis=1).round(1)

print("\n包含总分和平均分的数据：")
print(df.head())
```

### 4.2 计算班级排名

**任务**：按总分从高到低排个名。

```
# 按总分降序排列，并生成排名列
df['rank'] = df['total_score'].rank(method='first', ascending=False).astype(int)

# 按排名排序显示
df_sorted = df.sort_values('rank')
print("\n班级排名：")
print(df_sorted[['name', 'total_score', 'rank']])
```

### 4.3 计算各科及格率

**任务**：看看全班有多少人及格（>=60分）。

```
# 计算数学及格率
pass_math = (df['math'] >= 60).sum()
rate_math = pass_math / len(df) * 100
print(f"\n数学及格率：{rate_math:.1f}%")
```

## 📊 第五阶段：数据可视化（Matplotlib & Seaborn）

### 5.1 绘制各科平均分柱状图

**任务**：用条形图对比哪科最难。

**代码指引**：

```
import matplotlib.pyplot as plt
import seaborn as sns

# 设置中文字体（防止乱码）
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 准备数据
subjects = ['语文', '数学', '英语', '物理']

# 画图

```

### 5.3 绘制学生成绩雷达图（进阶）

**任务**：选一个同学，看看他的强项和弱项。

```
from math import pi

# 选择一个学生（比如张三）
student = df.iloc[0]
categories = subjects
values = student[['chinese', 'math', 'english', 'physics']].values

# 雷达图需要闭合，所以要首尾相连
values = list(values) + values[:1]
angles = [i * 2 * pi / len(categories) for i in range(len(categories))]
angles += angles[:1]

plt.figure(figsize=(6, 6))
ax = plt.subplot(111, polar=True)
plt.xticks(angles[:-1], categories)
ax.plot(angles, values)
ax.fill(angles, values, 'skyblue', alpha=0.5)
plt.title(f'{student["name"]} 的成绩雷达图')
plt.show()
```
