## 📌 项目简介
本项目是一个基于 Python 的**学生成绩数据分析与可视化系统**。旨在通过对班级或年级的考试成绩数据进行清洗、统计和绘图，帮助老师或教务人员快速发现教学薄弱环节，以及学生的偏科情况。

> **声明**：本项目为个人练习 Python 数据分析（EDA）的 Demo，主要目的是熟练掌握 Pandas、Matplotlib 等数据处理库的使用。

## ✨ 核心功能
- 📥 **数据导入**：支持读取 Excel/CSV 格式的成绩单。
- 🧹 **数据清洗**：自动处理缺失值、去除重复记录、格式化列名。
- 📈 **统计分析**：
  - 计算总分、平均分、最高分/最低分。
  - 计算班级排名和各科目及格率。
- 📊 **可视化图表**：
  - 绘制各科平均分柱状图。
  - 绘制成绩分布直方图（查看分数段人数）。
  - 绘制学生成绩雷达图（查看个人强弱项）。
- 💾 **结果导出**：将分析报告导出为新的 Excel 文件。

## 🛠️ 技术栈 (Tech Stack)
- **核心语言**：Python 3.9+
- **数据处理**：Pandas, NumPy
- **数据可视化**：Matplotlib, Seaborn
- **GUI 界面 (可选)**：Tkinter / PyQt5 (如果是命令行项目则去掉此项)
